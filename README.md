# SWEPetFirst

SWEPetFirst is a simple web-based pet first aid management system developed for the Software Engineering assignment.

## Dependencies

Install the required dependency before running the application:

```bash
pip install flask
```

## Project Setup

Navigate to the project root directory:

```bash
cd petfirstaid
```

## Run the Application

Start the Flask application:

```bash
python app.py
```

## Access the Application

Open the application in your browser:

```text
http://localhost:5000
```
